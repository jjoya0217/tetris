# 📝 업데이트 내역

## v2.0 - 리더보드 기능 추가 (2026-01-21)

### 🎉 새로운 기능

#### 1. 닉네임#번호 시스템
- 이름 입력 시 자동으로 4자리 번호 부여
- 예: "철수" 입력 → "철수#2847"로 표시
- 같은 이름도 구별 가능!
- 한 번 부여된 번호는 계속 유지됨

#### 2. 전체 순위표
- **최고 점수 순위**: 역대 최고 점수 기준
- **승리 횟수 순위**: 총 승리 게임 수 기준
- 상위 5명 표시
- 내 순위는 파란색으로 하이라이트

#### 3. 순위표 확인 위치
- **메인 화면**: "📊 순위표 보기" 버튼
- **게임 종료 화면**: 자동으로 순위표 표시

#### 4. 기록되는 정보
- 플레이어 이름 (닉네임#번호)
- 최고 점수
- 총 게임 수
- 총 승리 수
- 승률 (자동 계산)
- 마지막 플레이 날짜
- 최대 연속 승리

### 🔧 Firebase 보안 규칙 업데이트 필요!

기존 사용자는 Firebase Console에서 보안 규칙을 업데이트해야 합니다:

1. Firebase Console → Realtime Database → 규칙 탭
2. 다음 규칙으로 변경:

```json
{
  "rules": {
    "rooms": {
      "$roomId": {
        ".read": true,
        ".write": true
      }
    },
    "leaderboard": {
      ".read": true,
      "$playerId": {
        ".write": true
      }
    }
  }
}
```

3. "게시" 버튼 클릭

### 🆘 문제가 있나요?

업데이트 후 순위표가 안 보인다면:
1. Firebase 보안 규칙을 업데이트했는지 확인
2. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
3. 페이지 새로고침 (Ctrl+F5)

여전히 문제가 있다면 F12를 눌러 Console 탭에서 에러 메시지를 확인하세요!
